# 🌱 Automatic Irrigation System

An automatic irrigation system using **Arduino UNO** and **Soil Moisture Sensor**  
to automatically water plants when the soil becomes dry.

## ⚙️ Hardware Used
- Arduino UNO  
- Soil Moisture Sensor  
- Relay Module  
- DC Water Pump  
- Power Supply  
- Jumper Wires
